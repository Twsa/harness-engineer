    <p data-block-key="6ny7o">An AI agent is only as capable as the systems it can interact with. An agent that can &#x27;think&#x27; is great, but an agent that can manage code repositories, trigger workflows, or query databases is transformative.</p><p data-block-key="4j805">This shift from chat to action is exactly what <a href="https://google.github.io/adk-docs/">Agent Development Kit (ADK)</a> was built for. As an open-source framework, ADK provides the flexible primitives developers need to build, test, and deploy these kinds of complex, production-grade agentic workflows.</p><p data-block-key="80e3q">That’s why we are thrilled to announce a significant expansion to the Agent Development Kit (ADK) ecosystem. We have partnered with leading developer platforms to bring a broad suite of third-party tools and integrations directly into ADK.</p><p data-block-key="d75ko">With just a few lines of code, these integrations allow your agents to interact seamlessly with the real world. For example, watch how easily an agent can search Hugging Face to discover models and generate code:</p>

data-block-key="wtrrt" id="code-and-development"><b>Code &amp; development</b></h2><p data-block-key="dkcj2">Agents can connect directly into your development tools to help you manage code, test APIs, and execute scripts in secure environments:</p><ul><li data-block-key="ajo94"><a href="https://google.github.io/adk-docs/integrations/daytona/"><b>Daytona</b></a><b>:</b> Execute code, run shell commands, and manage files in isolated sandboxes.</li><li data-block-key="2gh05"><a href="https://google.github.io/adk-docs/integrations/github/"><b>GitHub</b></a><b>:</b> Analyze code, manage issues and PRs, and automate workflows.</li><li data-block-key="kq8f"><a href="https://google.github.io/adk-docs/integrations/gitlab/"><b>GitLab</b></a><b>:</b> Perform semantic code search, inspect CI/CD pipelines, and manage merge requests.</li><li data-block-key="9ao39"><a href="https://google.github.io/adk-docs/integrations/postman/"><b>Postman</b></a><b>:</b> Manage API collections, test APIs, and generate client code.</li><li data-block-key="76g4h"><a href="https://google.github.io/adk-docs/integrations/restate/"><b>Restate</b></a><b>:</b> Resilient agent execution with durable sessions and automatic recovery.</li></ul><h2 data-block-key="ek30j" id="project-management"><b>Project management</b></h2><p data-block-key="59gj4">Give your agents access to your team&#x27;s project tracking and documentation:</p><ul><li data-block-key="d0go1"><a href="https://google.github.io/adk-docs/integrations/asana/"><b>Asana</b></a><b>:</b> Manage projects, tasks, and goals for team collaboration.</li><li data-block-key="jbrj"><a href="https://google.github.io/adk-docs/integrations/atlassian/"><b>Atlassian</b></a><b>:</b> Manage Jira issues, search Confluence pages, and update team content.</li><li data-block-key="7erlb"><a href="https://google.github.io/adk-docs/integrations/linear/"><b>Linear</b></a><b>:</b> Manage issues, track project cycles, and streamline development workflows.</li><li data-block-key="833j9"><a href="https://google.github.io/adk-docs/integrations/notion/"><b>Notion</b></a><b>:</b> Search workspaces, create pages, and manage tasks and databases.</li></ul><h2 data-block-key="qudvh" id="databases"><b>Databases</b></h2><p data-block-key="ap22i">Connect your agents to databases and vector search engines:</p><ul><li data-block-key="5emdl"><a href="https://google.github.io/adk-docs/integrations/chroma/"><b>Chroma</b></a><b>:</b> Store documents and retrieve information using semantic search and metadata filtering.</li><li data-block-key="3hbgq"><a href="https://google.github.io/adk-docs/integrations/mongodb/"><b>MongoDB</b></a><b>:</b> Query collections, manage databases, and analyze schemas.</li><li data-block-key="3r0hk"><a href="https://google.github.io/adk-docs/integrations/pinecone/"><b>Pinecone</b></a><b>:</b> Store data, perform semantic search, and rerank results across vector indexes.</li></ul><h2 data-block-key="odtwp" id="memory"><b>Memory</b></h2><p data-block-key="469bv">Give your agents persistent, searchable memory across conversations:</p><ul><li data-block-key="5miev"><a href="https://google.github.io/adk-docs/integrations/goodmem/"><b>GoodMem</b></a><b>:</b> Automatic or agent-controlled semantic memory with multimodal document support.</li><li data-block-key="1ek8h"><a href="https://google.github.io/adk-docs/integrations/qdrant/"><b>Qdrant</b></a><b>:</b> Semantic memory layer for storing and retrieving information with data-block-key="f2mod"><a href="https://google.github.io/adk-docs/integrations/agentops/"><b>AgentOps</b></a><b>:</b> Session replays, metrics, and monitoring for ADK agents.</li><li data-block-key="ftkju"><a href="https://google.github.io/adk-docs/integrations/arize-ax/"><b>Arize AX</b></a><b>:</b> Production-grade observability, debugging, and evaluation of LLM applications.</li><li data-block-key="1f33o"><a href="https://google.github.io/adk-docs/integrations/freeplay/"><b>Freeplay</b></a><b>:</b> End-to-end observability, prompt management, and evaluation for AI agents.</li><li data-block-key="16g5g"><a href="https://google.github.io/adk-docs/integrations/mlflow/"><b>MLflow</b></a><b>:</b> Ingest OpenTelemetry traces for agent runs, tool calls, and model requests.</li><li data-block-key="csghi"><a href="https://google.github.io/adk-docs/integrations/monocle/"><b>Monocle</b></a><b>:</b> Open-source observability, tracing, and debugging of LLM applications.</li><li data-block-key="bg2f8"><a href="https://google.github.io/adk-docs/integrations/phoenix/"><b>Phoenix</b></a><b>:</b> Open-source, self-hosted observability, tracing, and evaluation of LLM applications.</li><li data-block-key="q5u2"><a href="https://google.github.io/adk-docs/integrations/weave/"><b>W&amp;B Weave</b></a><b>:</b> Log, visualize, and analyze model calls and agent performance.</li></ul><h2 data-block-key="szhoo" id="connectors"><b>Connectors</b></h2><p data-block-key="d95r9">Connect your agents to external apps and services through integration platforms:</p><ul><li data-block-key="ed74h"><a href="https://google.github.io/adk-docs/integrations/n8n/"><b>n8n</b></a><b>:</b> Trigger automated workflows, connect apps, and process data.</li><li data-block-key="6ddgj"><a href="https://google.github.io/adk-docs/integrations/stackone/"><b>StackOne</b></a><b>:</b> Connect agents to 200+ SaaS providers through a unified integration gateway.</li></ul><h2 data-block-key="sn7z3" id="ai-models-and-datasets"><b>AI models &amp; datasets</b></h2><p data-block-key="f1tof">Give your agents direct access to the AI/ML ecosystem:</p><ul><li data-block-key="69tm1"><a href="https://google.github.io/adk-docs/integrations/hugging-face/"><b>Hugging Face</b></a><b>:</b> Access models, datasets, research papers, and thousands of Gradio AI applications.</li></ul><h2 data-block-key="jgjqo" id="payments"><b>Payments</b></h2><p data-block-key="bfvv4">Let your agents work with you to handle your financial operations and billing:</p><ul><li data-block-key="4kk7d"><a href="https://google.github.io/adk-docs/integrations/paypal/"><b>PayPal</b></a><b>:</b> Manage payments, send invoices, and handle subscriptions.</li><li data-block-key="ej4ld"><a href="https://google.github.io/adk-docs/integrations/stripe/"><b>Stripe</b></a><b>:</b> Manage payments, customers, subscriptions, and invoices.</li></ul><h2 data-block-key="i3jeu" id="speech-and-audio"><b>Speech &amp; audio</b></h2><p data-block-key="7q5oq">Add voice and audio capabilities to your agents:</p><ul><li data-block-key="bq842"><a href="https://google.github.io/adk-docs/integrations/cartesia/"><b>Cartesia</b></a><b>:</b> Generate speech, localize voices across languages, and create audio content.</li><li data-block-key="aqeac"><a href="https://google.github.io/adk-docs/integrations/elevenlabs/"><b>ElevenLabs</b></a><b>:</b> Generate speech, clone voices, transcribe audio, and create sound effects.</li></ul><h2 data-block-key="injqq" id="email-and-messaging"><b>Email and messaging</b></h2><p data-block-key="3l4b6">Let your agents send, receive, and manage email:</p><ul><li data-block-key="vt0d"><a href="https://google.github.io/adk-docs/integrations/agentmail/"><b>AgentMail</b></a><b>:</b> Dedicated email inboxes for agents to manage conversations and attachments.</li><li data-block-key="e9rmb"><a href="https://google.github.io/adk-docs/integrations/mailgun/"><b>Mailgun</b></a><b>:</b> Send emails, track delivery metrics, and manage mailing lists.</li></ul><p data-block-key="89c6b">ADK also includes built-in integrations with Google Cloud services including BigQuery, Spanner, Pub/Sub, and more. See the full list in the <a href="https://google.github.io/adk-docs/integrations/">ADK Tools and Integrations</a> documentation.</p><h2 data-block-key="i2r85" id="a-unified-developer-experience"><b>A unified developer experience</b></h2><p data-block-key="bmkql">Your agent&#x27;s core logic shouldn&#x27;t be coupled to its tools. By using the <a href="https://google.github.io/adk-docs/tools-custom/mcp-tools/">McpToolset</a> primitive or <a href="https://google.github.io/adk-docs/plugins/">plugin</a> architecture, you can add any of these third-party integrations into your agent with just a <b>few lines of code</b>. ADK allows you to experiment and find the best tool for the job without ever having to refactor your agent. You don&#x27;t need to worry about the underlying implementation details - just copy the configuration for the tool you want, and the ADK handles the rest.</p><p data-block-key="frtb0">Here is an example of adding the GitHub integration to an ADK agent in Python, which instantly helps you empower your agent to manage repositories and automate workflows:</p>
</div>  <div class="inner-block-content code-block line-numbers">
    <pre><code class="language-python">from google.adk.agents import Agent
from google.adk.tools.mcp_tool import McpToolset
from google.adk.tools.mcp_tool.mcp_session_manager import StreamableHTTPServerParams

GITHUB_TOKEN = &quot;YOUR_GITHUB_TOKEN&quot;

# Initialize your agent
root_agent = Agent(
    model=&quot;gemini-3-flash-preview&quot;,
    name=&quot;github_agent&quot;,
    instruction=&quot;Help users get information from GitHub&quot;,
    # Add a third-party integration by configuring McpToolset
    tools=[
        McpToolset(
            connection_params=StreamableHTTPServerParams(
                url=&quot;https://api.githubcopilot.com/mcp/&quot;, 
                headers={
                    &quot;Authorization&quot;: f&quot;Bearer {GITHUB_TOKEN}&quot;,
                    &quot;X-MCP-Toolsets&quot;: &quot;all&quot;,
                    &quot;X-MCP-Readonly&quot;: &quot;true&quot;
                },
            ),
        )
    ],
)</code></pre>
